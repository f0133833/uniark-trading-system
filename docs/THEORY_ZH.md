# 交易体系的公理化构建

> 一份按照公理化方式整理的价格行为交易框架。借鉴欧几里得几何的层级演绎结构,
> 每一层只引入最少的新词汇,所有高层概念都由低层原语构造而来。
>
> **本文档是理论规范,不是交易建议、不是收益承诺、不是经过回测验证的策略。**
> 项目意图见 [README](../README_zh.md)。

---

## 一、公理层

**公理。** 市场不存在永恒状态,在任意给定时间尺度下,状态变化是反复发生的。

**定理。** 在任意给定时间尺度下,市场至少存在三段连续走势结构
S1→S2→S3(正反正,或者反正反)。

---

## 二、结构层

**定义【转折点】** 走势中将方向相反的连续段分隔开的价格点。

**定义【一段走势】** 从一个转折点到下一个转折点之间的价格路径。

**定义【三段结构】** 三段连续走势 S1→S2→S3
构成的结构,其中相邻段方向相反。

**定义【重叠区间】** 三段结构 S1、S2、S3 中:

- 取三段各自高点中的**最低值**为区间上界
- 取三段各自低点中的**最高值**为区间下界

上界与下界之间的价格区间即为重叠区间。

---

## 三、状态层

### 基础状态【盘整】

成立条件:

1. 包含三段及以上走势,且
2. 前三段存在重叠区间。

**盘整区间**:盘整结构中,前三段连续走势的重叠部分。

> **锚点不变性。** 盘整区间是一个固定锚点,由前三段确定,后续走势即使
> 延伸,盘整区间不变。

### 盘整之间的关系

- **盘整抬高**:前后两个盘整,且后盘整区间的低点 > 前盘整区间的高点。
- **盘整下降**:前后两个盘整,且后盘整区间的高点 < 前盘整区间的低点。

### 复合状态【上涨 / 下跌】

- **上涨**:包含至少两个盘整结构,且盘整抬高。
- **下跌**:包含至少两个盘整结构,且盘整下降。

**关键约束:** 前后盘整*结构*可以有走势重叠,但前后盘整*区间*必须无交集。

---

## 四、转折层

### 定义【盘整破坏】

旧盘整被破坏,当且仅当以下三条同时成立:

1. **触发**:一段走势离开旧盘整区间。
2. **完成**:该走势参与构建了新的三段结构。
3. **确认**:新盘整区间与旧盘整区间无交集。

**破坏方向判定:**

- 新盘整区间低点 > 旧盘整区间高点 → 盘整抬高 → 转折为上涨
- 新盘整区间高点 < 旧盘整区间低点 → 盘整下降 → 转折为下跌

### 定义【上涨破坏】

在上涨状态中,出现一段向上的走势突破最后一个盘整区间,但新三段结构构建
过程中价格与旧盘整区间产生重叠(构建失败)。

**效果:** 上涨破坏成立,转折为盘整或者下跌
(即 上涨→上涨+盘整 或 上涨→上涨+下跌)。

### 定义【下跌破坏】

在下跌状态中,出现一段向下的走势跌破最后一个盘整区间,但新三段结构构建
过程中价格与旧盘整区间产生重叠(构建失败)。

**效果:** 下跌破坏成立,转折为盘整或者上涨
(即 下跌→下跌+盘整 或 下跌→下跌+上涨)。

---

## 五、预测层

### 定义【力度】

一段走势 Si 的力度 = 该走势对应时间范围内,对应的 MACD 柱线
面积的绝对值之和:

> 力度(Si) = Σ |hist(t)|, 其中 t ∈ Si

### 定义【同向段】

设三段结构 S1→S2→S3,若 S1 与
S3 同方向,S2 为中间反向连接,则 S1 与
S3 是同向段。

### 定义【力度衰竭】

对比同向段 S3 和 S1 的力度,满足:

> 力度(S3) < 力度(S1) × 50%

> 50% 是经验参数,在不同市场、不同时间尺度下可以做调整。

### 定义【背离】

比较同向段 S1、S3,若力度衰竭成立,且 S3
相对 S1 创出价格新高(上涨段)或者新低(下跌段),则称为背离。

- **趋势背离(强)**:S1 与 S3 之间由*盘整结构*
  S2 连接,且背离成立。
- **三段背离(弱)**:S1 与 S3 之间由*单一反向段*
  S2 连接,且背离成立。

## 六、执行层

> **提醒。** 以下信号是结构变化的理论位置,不是交易指令,也未在本项目中
> 经过回测验证。

### 破坏卖出

条件:

- 当前处于**上涨**状态。
- **趋势背离**成立,上涨被破坏。
- S3 向上突破 S2 盘整区间。
- 力度(S3) < 力度(S1) × 50%。

→ **卖出位置:** 新高价格附近。

### 破坏买入

对称于破坏卖出:

- 当前处于**下跌**状态。
- 趋势背离成立,下跌被破坏。
- S3 向下突破 S2 盘整区间。
- 力度(S3) < 力度(S1) × 50%。

→ **买入位置:** 新低价格附近。

### 盘整卖出

条件:

- S1 是下跌段,MACD 快慢线在 0 轴下方运行
  (或者有效跌破 0 轴)。
- 伴随构建盘整结构 S2(内部三段连续走势),MACD 快慢线
  回抽 0 轴。
- S2 构建完成,MACD 快慢线突破 0 轴失败。
- 力度(S3) ≥ 力度(S1) × 50%。

→ **卖出位置:** 下跌段 S3 的起点附近。

### 盘整买入

对称于盘整卖出:

- S1 是上涨段,MACD 快慢线在 0 轴上方运行
  (或有效突破 0 轴)。
- 伴随构建盘整结构 S2(内部三段连续走势),MACD 快慢线
  回踩 0 轴。
- S2 构建完成,MACD 快慢线再次从 0 轴拉起。
- 力度(S3) ≥ 力度(S1) × 50%。

→ **买入位置:** S3 上涨起点附近。

## 关于范围与局限

本框架是描述价格行为的**语言**,不是预测保证。它的断言都是有条件的:

- 它不断言任何市场总是表现出清晰的三段结构,只断言在任意时间尺度下
  *至少存在一个*这样的结构。
- 它不主张背离一定会引发价格反转,只是标定了"反转候选位置"成立的结构条件。
- 50% 的力度衰竭阈值是经验值,不是推导出来的;用户在不同品种和时间
  尺度下可以合理调参。
- 特殊市场状况(极端单边趋势、流动性枯竭的市场、政策性熔断、结构性
  断点等)可能使框架的根本假设——包括存在性公理——失效。

这份工作的贡献在于使框架**可证伪**:每一条断言都精确到可以被测试、
反驳、修正的程度。这是任何"交易体系"成为理性工具(而非信仰)的必要前提。
